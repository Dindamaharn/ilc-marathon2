Profil

Komunitas ILC (Informatics Learning Community) yang didirikan di Fakultas Ilmu Komputer UPN Veteran Jawa Timur. Bertujuan sebagai tempat untuk belajar bersama para mahasiswa Fasilkom pada pengembangan Web Developer. Kami menyediakan berbagai program belajar. Mulai dari program Bootcamp hingga webinar yang informatif.

Kegiatan

Kegiatan ILC (Informatics Learning Community) memiliki beragam kegiatan, mulai dari bootcamp web development, webinar linux yang terbuka untuk umum ataupun anggota ILC itu sendiri, hingga tanya jawab (QnA) tentang seputar Web development dan masih banyak event event ataupun kegiatan yang dapat mengembangkan para mahasiswa di fakultas ilmu komputer UPN Veteran Jawa Timur
